    I am currently working on a bachelor’s degree in computer science. Recently, I’ve been

    expanding my experience with methods for parallel programming and utilizing GPUs for

    code acceleration though an internship at Oak Ridge National Lab at the OLCF. As part

    of my internship I’ve written code that I ran on the Titan supercomputer and written

    three tutorials that go with it. These tutorials show the parallelization of code using MPI,

    and OpenMP, OpenACC, CUDA, and Thrust. The tutorials I developed were a Monte Carlo

    integrator that calculates pi, and a code that will generate the Mandelbrot set in parallel. My

    next project is a MAGMA-related tutorial to Fortran and C. This tutorial will also encompass other linear

    algebra libraries, mainly CULA and LAPACK.

    Recently I gave a seminar at the University of Tennessee on the subject of

    accelerating serial code for the GPU. I’ve also given talks to groups of

    high school and early college students about the basic principles of High Performance

    Computing. In my free time, I’ve built two 4-node clusters. I’ve always been interested in

    computing but by building the first cluster, it pushed me to research more into and focus my

    interest into HPC related areas.
